# Day35 - Smart Vehicle Safety System

## Objective
To design a smart vehicle safety system that detects obstacles, controls headlights, and applies automatic braking.

## Platform
- Wokwi Simulation
- Arduino UNO

## Components Used
- Ultrasonic Sensor
- Potentiometer (LDR simulation)
- LED (Headlight)
- Buzzer (Warning)
- Servo Motor (Brake system)

## Working Principle
1. Ultrasonic sensor measures distance
2. If obstacle is near → buzzer ON + brake applied
3. Light sensor checks environment
4. If dark → headlight ON
