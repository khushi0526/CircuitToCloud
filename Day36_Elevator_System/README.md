# Day36 - Smart Elevator System

## Objective
To simulate an elevator system that moves between floors based on user input.

## Platform
- Wokwi Simulation
- Arduino UNO

## Components Used
- Push Buttons (Floor selection)
- LEDs (Floor indicators)
- Buzzer
- Servo Motor (door)

## Working Principle
1. User presses a floor button
2. Elevator moves step-by-step
3. LED indicates current floor
4. On reaching:
   - Buzzer alerts
   - Door opens and closes
