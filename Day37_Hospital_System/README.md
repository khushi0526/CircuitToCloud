# Day37 - Smart Hospital Patient Monitoring System

## Objective
To monitor patient health parameters and trigger alerts in abnormal conditions.

## Platform
- Wokwi Simulation
- Arduino UNO

## Components Used
- DHT22 Sensor
- Potentiometer (Pulse simulation)
- LCD (I2C)
- LED
- Buzzer

## Working Principle
1. Sensor reads temperature and pulse
2. Data displayed on LCD
3. System checks thresholds
4. If abnormal → alert triggered
