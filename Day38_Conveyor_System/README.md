# Day38 - Smart Factory Conveyor System

## Objective
To design an automated conveyor system that detects objects and controls movement.

## Platform
- Wokwi Simulation
- Arduino UNO

## Components Used
- IR Sensor
- Push Button
- Servo Motor
- LED
- Buzzer

## Working Principle
1. User turns system ON using button
2. Conveyor runs continuously
3. If object detected:
   - Conveyor stops
   - Alert triggered
4. When object removed → resumes
